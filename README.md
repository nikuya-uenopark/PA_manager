## 🚀 セットアップ

### 必要な環境

- Node.js 16.0以降
- MySQL 5.7以降
- npm または yarn

### 1. リポジトリのクローン

```bash
git clone https://github.com/nikuya-uenopark/PA_manager.git
cd PA_manager
```

### 2. 依存関係のインストール

```bash
npm install
```

### 3. 環境変数の設定

```bash
cp .env.example .env
# .envファイルを編集してデータベース設定を入力
```

### 4. データベースのセットアップ

```bash
npm run setup-db
```

### 5. サーバーの起動

```bash
# 開発モード
npm run dev

# 本番モード
npm start
```

## 使い方

1. **スタッフの追加**
   - 「スタッフ追加」ボタンから新しいスタッフを登録
   - 名前と役職を入力

2. **評価項目の設定**
   - 「項目追加」ボタンから評価項目を追加
   - カテゴリーと説明を設定

3. **評価の実施**
   - スタッフカードをクリックして詳細画面を開く
   - 各項目を「できる」「学習中」「できない」で評価

4. **進捗の確認**
   - スタッフカードの進捗バーで一目で状況を把握
   - パーセンテージで習得率を表示
